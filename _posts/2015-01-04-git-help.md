## Tag

- `git tag` 列出已有标签
- `git tag v0.1` 新建标签
- `git show v0.1` 查看标签版本信息
- `git push origin v0.1` 部署 `v0.1` 标签
- `git push origin --tags` 部署所有本地标签
- `git tag -d v0.2` 删除标签
- `git push origin :refs/tags/v0.2` 应用删除操作
